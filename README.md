# Telegram Weather Bot on Vercel

Этот проект представляет собой Telegram-бота на Node.js и TypeScript, готового к деплою в Vercel как Serverless Function.
Бот каждый день проверяет погоду в селе Лукьяновка (Белоцерковский район), анализирует её с помощью ИИ (Gemini) и отправляет суровую рекомендацию по экипировке.

## 🚀 Настройка и Деплой в Vercel

### 1. Подготовка переменных окружения (ENV)
Для работы бота необходимо добавить следующие переменные окружения в настройках проекта на Vercel (раздел **Settings -> Environment Variables**):

*   **`TELEGRAM_BOT_TOKEN`**: Токен вашего бота. 
    *   *Как получить:* Напишите боту [@BotFather](https://t.me/BotFather) в Telegram, создайте нового бота (`/newbot`) и скопируйте предоставленный токен.
*   **`CHAT_ID`**: Ваш личный идентификатор чата в Telegram.
    *   *Как получить:* Напишите боту [@userinfobot](https://t.me/userinfobot) или [@getmyid_bot](https://t.me/getmyid_bot) в Telegram. В ответ он пришлет ваш `ID` (это число, например, `123456789`). Укажите именно его.
*   **`OPENWEATHER_API_KEY`**: API ключ от сервиса погоды.
    *   *Как получить:* Зарегистрируйтесь на сайте [OpenWeatherMap](https://openweathermap.org/) и сгенерируйте API ключ в личном кабинете.
*   **`GEMINI_API_KEY`**: API ключ для нейросети Google Gemini.
    *   *Как получить:* Получите ключ в [Google AI Studio](https://aistudio.google.com/app/apikey).

### 2. Запуск Vercel Cron
Запуск бота настроен через `vercel.json` по расписанию:
*   Расписание (CRON): `0 4 * * *` (04:00 по UTC)
*   **Важно:** Vercel Cron работает исключительно по времени UTC. 04:00 по UTC соответствует 07:00 по Киевскому времени (летом, UTC+3) и 06:00 (зимой, UTC+2).

### 3. Деплой
1. Запушьте код в репозиторий GitHub.
2. В панели управления Vercel создайте новый проект и выберите этот репозиторий.
3. Добавьте все переменные окружения.
4. Нажмите **Deploy**.

### 4. Активация Webhook (Интерактивные команды)
Чтобы бот отвечал на твои сообщения в Telegram в реальном времени, нужно связать Telegram с твоим сервером Vercel. 
После успешного деплоя скопируй эту ссылку, вставь в браузер и нажми Enter:

`https://api.telegram.org/bot<ТВОЙ_TELEGRAM_BOT_TOKEN>/setWebhook?url=https://<твой-домен-в-vercel>.vercel.app/api/webhook`

*   Замени `<ТВОЙ_TELEGRAM_BOT_TOKEN>` на токен твоего бота.
*   Замени `<твой-домен-в-vercel>.vercel.app` на реальный домен твоего проекта.

Если всё пройдет успешно, браузер покажет: `{"ok":true,"result":true,"description":"Webhook was set"}`.

## Тестирование
После настройки вебхука открой бота в Telegram:
1. Напиши команду `/now` — бот соберет данные, сгенерирует картинку и пришлет сводку мгновенно.
2. Напиши любое сообщение (например, "Как дела?") — бот ответит тебе в своем суровом стиле.
3. Утренняя сводка (Cron) продолжит работать автоматически в 07:00.
